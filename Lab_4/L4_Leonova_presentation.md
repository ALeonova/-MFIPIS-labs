# Теоретическое введение

## Алгоритм Евклида

+ НОД(A,0) = A
+ НОД(0,B) = B
+ Если A = B*Q + R и B≠0, то НОД(A, B) = НОД(B, R), 

где Q — целое число, а R — целое число от 0 до B-1

## Бинарный алгоритм Евклида

Основан на использовании следующих свойств НОД:

+ НОД(2m, 2n) = 2 НОД(m, n),
+ НОД(2m, 2n+1) = НОД(m, 2n+1),
+ НОД(-m, n) = НОД(m, n)
